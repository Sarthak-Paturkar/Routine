# Routine
A simple and practical **Daily Routine Tracker** designed to help users build better habits and stay consistent with their daily goals. The tracker focuses on two main parts: **Daily Tracking**, which is mandatory, and an optional **21-Day Challenge** for users who want to push themselves to build a new habit.

In the **Daily Tracking** section, users can mark their progress on common productivity and self-improvement activities such as cardio, spirituality or meditation, studies, coding, reading, journaling, healthy eating, sleep tracking, and skill learning. To keep it flexible, there is also an **“Other” option** that allows users to add and track their own custom activities.

The **21-Day Challenge** section is optional and allows users to select a habit they want to practice consistently for 21 days. It includes simple progress tracking so users can see how far they’ve come and stay motivated to complete the challenge.

The goal of this project is to provide a **clean, minimal, and easy-to-use habit tracker** that encourages discipline, consistency, and personal growth through small daily actions.
